## Features

Initial stable release. The `vitest-agent` CLI: a `doctor` command, a `db` command group (`path`, `prune`, `reset`, `query`), and an `agent` namespace for hook-driven plumbing (`triage`, `wrapup`, `record`, `register-agent`, `end-agent`, `inject-env`, `sidecar-path`).
